📧 Spam Email Classifier using Machine Learning
📖 Overview

This project is a Spam Email Classifier that detects whether a message is Spam 🚫 or Not Spam ✅ using Machine Learning techniques. It uses Natural Language Processing (NLP) and the Naive Bayes algorithm for accurate classification.

🚀 Features
🔍 Detects spam messages instantly
📊 High accuracy (~95%+)
⚡ Fast and efficient
🤖 Uses TF-IDF for text processing
🖥️ Interactive UI using Gradio
🛠️ Technologies Used
Python
Pandas
Scikit-learn
Natural Language Processing (TF-IDF)
Gradio (for prototype UI)
📂 Dataset
SMS Spam Collection Dataset
Contains labeled messages:
Spam
Ham (Not Spam)
⚙️ Workflow
Load dataset
Data preprocessing
Train-test split
Text vectorization using TF-IDF
Model training using Naive Bayes
Prediction and evaluation
🧠 Model Used
Multinomial Naive Bayes
Best suited for text classification
Works efficiently with word frequencies
📊 Results
Achieved accuracy of 95%+
Successfully classifies spam messages

Congratulations! You won a free iPhone
